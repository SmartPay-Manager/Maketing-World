## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A modern web browser

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Mialy333/DeFi-PulseX.git
cd DeFi-PulseX
```

2. Install dependencies:

```bash

```

3. Start the development server:

```bash
npm run dev
```
